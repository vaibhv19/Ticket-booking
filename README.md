Developed a ticket booking CLI program that handles booking logic, user management, and seat allocation using Core Java.

*Implemented booking validation and seat tracking logic

*Persisted data using JSON (no external database)

*Applied OOP principles and separation of concerns

*Used Java Streams and Collections for efficient data handling

Tech Stack:
Java, Gradle, File I/O, Jackson (ObjectMapper), Collections Framework

This project strengthened my understanding of backend architecture and real-world application flow without relying on frameworks.
